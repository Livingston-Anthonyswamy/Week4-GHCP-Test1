# Test Plan for COBOL Student Account Management System

This test plan covers the business logic of the COBOL-based student account management system. It includes manual test cases for validating functionality before transformation to a Node.js application. The plan focuses on account operations: viewing balance, crediting, debiting, and menu navigation.

| Test Case ID | Test Case Description | Pre-conditions | Test Steps | Expected Result | Actual Result | Status (Pass/Fail) | Comments |
|--------------|-----------------------|----------------|------------|----------------|----------------|---------------------|----------|

| TC002 | Credit Account with Valid Amount | Application is running and displaying main menu. Initial balance is 1000.00. | 1. Select option 2 (Credit Account).<br>2. Enter a valid amount (e.g., 500.00) when prompted. | Balance increases by the entered amount. Displays "Amount credited. New balance: 001500.00" and returns to menu. |  |  | Test positive credit operation. |
